## E. coli 16S rRNA Bioinformatics Analysis


## Project Objectives
- Retrieve 16S rRNA sequences from NCBI using the `rentrez` package  
- Perform sequence cleaning and preprocessing  
- Calculate GC content  
- Conduct multiple sequence alignment (MSA)  
- Generate a phylogenetic tree to compare sequence similarities  
- Organize outputs into a reproducible project structure  

## Tools & Packages Used
- **rentrez** – Access NCBI databases  
- **Biostrings** – Sequence handling and manipulation  
- **msa** – Multiple sequence alignment  
- **ape** – Phylogenetic analysis  
- **tidyverse** – Data wrangling
